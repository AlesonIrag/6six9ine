# Color Variants - Quick Admin Guide 🎨

## How to Add Products with Color Variants

### Step 1: Upload Images
```
Click "Choose Images" → Select all color variant photos
```
Example: Upload 3 photos (Black shirt, White shirt, Navy shirt)

### Step 2: Select Colors
For each uploaded image, you'll see a dropdown menu:

## Tips for Best Results

### ✅ DO:
- Upload clear, well-lit photos
- Use same angle for all color variants
- Select accurate color names
- Verify all colors before saving

### ❌ DON'T:
- Leave color dropdowns empty
- Mix different product angles
- Use poor quality images
- Forget to save after selecting colors

## Need Help?

### Color Not in List?
Choose the closest match. For example:
- "Slate" → Use **Charcoal** or **Dark Gray**
- "Emerald" → Use **Green** or **Forest Green**
- "Crimson" → Use **Red**
- "Aqua" → Use **Cyan** or **Turquoise**

### Color Swatch Wrong?
Make sure you selected the correct color from the dropdown. The swatch updates automatically when you change the selection.

### Can't See Dropdown?
Upload at least one image first. The color dropdown appears after images are uploaded.
